# 持久战阶段评估指南

配合 `protracted-strategy` skill 使用。帮助判断长期任务当前所处阶段，选择对应策略。

## 三阶段速查

| 阶段 | 核心特征 | 正确策略 | 错误策略 |
|------|---------|---------|---------|
| 战略防御 | 信息不足，能力有限，处于探索期 | 保存实力，调查研究，积累认识 | 急于全面出击 |
| 战略相持 | 有一定基础但未达突破条件，最困难 | 积小胜为大胜，耐心推进 | 消极等待或冒进 |
| 战略反攻 | 条件成熟，优势积累到位 | 果断推进，全面展开 | 犹豫不决错过时机 |

## 阶段判断指标

### 你处于**战略防御**阶段的信号

- [ ] 对问题域的理解还很表面，经常发现新的未知
- [ ] 尝试的方案频繁失败或需要大幅调整
- [ ] 没有可靠的基础设施/工具链可以依赖
- [ ] 团队/个人对任务的信心不足
- [ ] 外部约束（需求不明、依赖缺失）限制了行动空间

**此阶段应该做的：**
- 大量调查研究（`investigation-first`）
- 建立最小可用的根据地（`spark-prairie-fire`）
- 允许战略退却——聪明的拳师先退一步
- 不追求完美，追求"站稳脚跟"

### 你处于**战略相持**阶段的信号

- [ ] 已有初步的认知和基础，但离目标仍有明显距离
- [ ] 进展存在但缓慢，容易感到焦虑或疲惫
- [ ] 遇到了一些"硬骨头"问题尚未攻克
- [ ] 外部条件暂时不会有大的变化
- [ ] 已积累的经验开始发挥作用，但尚不足以决定性突破

**此阶段应该做的：**
- 在每个具体问题上集中兵力速决（`concentrate-forces`）
- 积小胜为大胜，保持推进节奏
- 这是"最困难的时期，然而它是转变的枢纽"——坚持
- 持续观察转折点信号

### 你处于**战略反攻**阶段的信号

- [ ] 对问题的理解已经深入，方案清晰可靠
- [ ] 基础设施和工具已经到位，执行效率高
- [ ] 主要障碍已被消除或大幅削弱
- [ ] 团队/个人信心充足，执行力强
- [ ] 外部条件有利，窗口期出现

**此阶段应该做的：**
- 果断全面推进，不犹豫
- 保持势头，连续作战
- 但仍然战术上重视每一个细节
- 不因接近胜利而放松

## 阶段转化的关键信号

### 防御 → 相持
- 对问题有了系统性认知（不再频繁遇到"意外"）
- 有了可靠的基础设施/根据地
- 能稳定地解决局部问题

### 相持 → 反攻
- 积累的小胜利量变引发质变
- 关键障碍被突破
- 外部条件出现有利变化
- 执行信心和能力达到临界点

### 注意：转化不是瞬间的
> "第二阶段是整个战争的过渡阶段"

阶段转化是渐进的。不要期待一个明确的"现在可以反攻了"的信号。持续评估，当多个指标同时满足时，就是时机。

## 常见误判

| 误判 | 实际情况 | 纠正 |
|------|---------|------|
| "已经可以反攻了"（实际仍在相持） | 被一两次小胜冲昏头脑 | 检查是否满足多个反攻指标 |
| "还在防御期"（实际已进入相持） | 过度谨慎，不敢推进 | 评估是否已有稳定的根据地 |
| "没有进展"（实际在相持期正常推进） | 对相持期的缓慢进展不耐烦 | 相持期本身就是最困难也最关键的 |
